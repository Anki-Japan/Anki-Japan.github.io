  1. object Constants

    Contains keys for the map of orders.

    Contains keys for the map of orders. The map of orders contains order keys (VOCABULARY, KANJI, RADICALS, MEDIA, EXCLUDE) and the order specifications as the values. For example:
    Map(
    VOCABULARY -> s"$I $M $R $E"
    )

  2. object ProcessOrder

    Processes the orders for notes generation.

    Processes the orders for notes generation. Results in the generation of a Map, with the Constants object. Think of it as of a directory with files. It is supposed to be converted into it.

    Constants for more information on composing orders.
